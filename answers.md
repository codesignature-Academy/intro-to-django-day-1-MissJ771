# Django Assignment Answers

## Question 1

### manage.py
Its the script that configures the django project I'm working on and executes the command line instructions like runserver, etc.

### settings.py
This is the control center of my project that tells my project how to operate which includes the installed apps, database, etc

### urls.py
This one is incharge of directing incoming URLs to the appropriate views so django will know how to handle them.
